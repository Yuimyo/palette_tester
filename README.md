# palette_tester

パレットの作成を支援するソフトウェアです。
パレットのスクラップ＆ビルドが簡単に行え、パレットを自由自在にテストできます。

ブラウザ上で動作します。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
